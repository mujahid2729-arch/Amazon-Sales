# Amazon-Sales
Amazon Sales Analysis dashboard built using Power BI to track YTD and QTD sales, products sold, and customer reviews. It includes interactive visuals for monthly and weekly trends, product categories, and top-performing products to support data-driven business insights.

Amazon Sales Analysis Dashboard (Power BI)

OVERVIEW
This project analyzes Amazon product sales data using Power BI. The dashboard provides insights into sales performance, product trends, and customer engagement through interactive visualizations.

KEY FEATURES
YTD and QTD Sales analysis
Products sold and customer reviews tracking
Monthly and weekly sales trends
Sales by product category
Top 5 products by sales and reviews
Interactive filters for better analysis
Tools Used
Power BI
Power Query
DAX

PURPOSE
The goal of this project is to demonstrate data analysis and dashboarding skills by transforming raw sales data into meaningful business insights for decision-making
